========================================================================
TELC A1-A2 GRAMMATIKTRAINING: PRÄPOSITIONALADVERBIEN (WO- / DA-)
========================================================================

KURAL HATIRLATMASI (A1-A2 SEVİYESİ):
- Soru sorarken: "wo" + edat (Edat sesli harfle başlıyorsa araya -r- girer: wo-r-auf / wo-mit)
- Cevap verirken (Ona / Onunla): "da" + edat (Edat sesli harfle başlıyorsa araya -r- girer: da-r-auf / da-mit)

------------------------------------------------------------------------

ÜBUNG 1: Wo- oder Wor-? (Soru Kelimesini Seçin)
Ergänzen Sie die Fragen mit "Wo-" oder "Wor-".

1. ___________auf wartest du? Auf den Bus? (warten auf)
2. ___________mit fährst du nach Berlin? Mit dem Zug? (fahren mit)
3. ___________auf hast du Lust? Auf Pizza oder Pasta? (Lust haben auf)
4. ___________mit spielt das Kind? Mit dem Ball? (spielen mit)


ÜBUNG 2: Da- oder Dar-? (Kısa Cevap Yapısı)
Antworten Sie auf die Fragen mit "damit" oder "darauf".

1. Wartest du auf das Express-Paket?
   -> Ja, ich warte _______________.

2. Bist du mit dem neuen Fahrrad zufrieden?
   -> Ja, ich bin _______________ sehr zufrieden.

3. Hast du Lust auf einen Kaffee?
   -> Nein, ich habe keine Lust _______________.

4. Arbeitest du heute mit dem neuen Computer?
   -> Ja, ich arbeite _______________.


ÜBUNG 3: telc-Format (Mehrfachauswahl / Çoktan Seçmeli)
Welches Wort passt perfekt in die Lücke? Kreuzen Sie an (A, B oder C).

1. - _________ hast du Lust? - Auf ein Eis, es ist so heiß!
   [ ] A) Worauf
   [ ] B) Womit
   [ ] C) Daauf

2. Mein Handy ist alt, aber ich bin noch zufrieden _________.
   [ ] A) darauf
   [ ] B) damit
   [ ] C) wovon

3. - Wartet ihr auf den Handwerker? - Ja, wir warten schon zwei Stunden _________.
   [ ] A) damit
   [ ] B) wovor
   [ ] C) darauf


========================================================================
LÖSUNGEN (CEVAP ANAHTARI)
========================================================================

ÜBUNG 1:
1. Worauf ("auf" sesli ile başlar, araya 'r' girer)
2. Womit ("mit" sessiz ile başlar, direkt birleşir)
3. Worauf ("auf" sesli ile başlar, araya 'r' girer)
4. Womit ("mit" sessiz ile başlar, direkt birleşir)

ÜBUNG 2:
1. darauf (auf -> dar-auf)
2. damit (mit -> da-mit)
3. darauf (auf -> dar-auf)
4. damit (mit -> da-mit)

ÜBUNG 3:
1. A - "Lust haben auf" kalıbı için soru kelimesi sesli uyumuyla "Worauf" olur.
2. B - "Zufrieden sein mit" kalıbına atıf yapmak için "damit" (onunla/bundan) kullanılır.
3. C - "Warten auf" kalıbında "onu bekliyoruz" demek için "darauf" doğru seçenektir.
